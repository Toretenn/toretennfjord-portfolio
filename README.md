# Tore Tennfjord — Portefølje

Personlig porteføljenettside for Tore Tennfjord, Digital Designer & Markedsfører i Bergen.

Live: https://toretennfjord.no

## Om prosjektet
Statisk nettside bygget med ren HTML og CSS — ingen rammeverk, ingen byggeprosess.
Hostes gratis på Netlify med kontinuerlig publisering fra dette GitHub-repoet.

## Struktur
```
index.html                          Forside
arbeid.html                         Prosjektoversikt
om.html                             Om meg
kontakt.html                        Kontakt
prosjekt-valentine.html             Case: Valentinskampanje (Matarena)
prosjekt-plakater.html              Case: Plakatserie (reker & krabber)
prosjekt-havforskningsinstituttet.html  Case: Sosialt innhold (HI)
prosjekt-savr.html                  Case: Savr (eget produkt)
styles.css                          Felles stilark (design system)
assets/                             Bilder, video og CV
```

## Slik oppdaterer jeg siden
1. Rediger filene lokalt.
2. Commit og push til GitHub.
3. Netlify publiserer den nye versjonen automatisk på sekunder.

## Design system
- Fraunces (display/overskrifter) + Garet (brødtekst/UI)
- Primærfarge #E67810 · Bakgrunn #FFF9EC · Tekst #2D2D2D
